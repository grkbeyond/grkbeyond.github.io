ListView源码解析

发表于 2016-04-10 | 分类于 Android

ListView是一个以列表形式用来展示多条数据的Android原生控件，可以通过上下滑动来查看更多的数据。即使数据再多，也不会发生内存溢出的问题。

RecycleBin

ListView加载许多数据而不会导致内存溢出的最重要的功臣就是RecycleBin。是AbsListView内部类，这样AbsListView的所有子类都能有这个特性。

1，fillActiveViews()：保存指定View到mActiveViews中。
2，getActiveView()：根据position获取mActiveViews中相应的View。一旦被获取了之后就会从mActiveViews当中移除，下次获取同样位置的View将会返回null。
3，addScrapView()：将一个废弃的View缓存起来，RecycleBin当中使用mScrapViews和mCurrentScrap这两个List来存储废弃View。
4，getScrapView()：从缓存集合中获取一个View。

ListView首次展示数据。

主要从onLayout()方法作为切入点来了解ListView的工作机制。
直接看ListView的layoutChildren()方法。

layutChildren()

ListView刚开始绘制的时候还没有子View,所以在第几行中，getChildCount()方法返回的是0。
在第几行，数据源没有发生变化，dataChanged为false，所以执行recycleBin.fillActiveViews(childCount, firstPosition)方法，现在没有子view所以这行代码没有实质性作用。
接下来一般会执行默认的布局模式(LAYOUT_NORMAL)代码，也就是第几行的default。childCount为0且默认是从上而下布局子View的，所以执行fillFromTop(childrenTop)方法。

###fillFromTop(childrenTop)

其主要调用fillDown(mFirstPosition, nextTop)方法。

fillDown(mFirstPosition, nextTop)方法。

如果下一个子View的top小于ListView的bottom并且pos小于元素数量的时候执行while循环语句。当子view超出当前屏幕里或者pos大于mItemCount的时候跳出循环。
循环里主要的执行makeAndAddView(pos, nextTop, true, mListPadding.left, selected)。

makeAndAddView(int position, int y, boolean flow, int childrenLeft,boolean selected)

1，在第几行，数据源没有发生变化，dataChanged为false,所以先执行mRecycler.getActiveView(position);获取一个缓存的View。之前在layoutChildren()方法中并没有缓存View所以返回的是null。
2，接着执行第几行的代码。obtainView(position, mIsScrap);
3，最后执行setupChild(child, position, y, flow, childrenLeft, selected, mIsScrap[0]);

obtainView(int position, boolean[] isScrap)

1，在obtainView方法中先在第几行执行mRecycler.getScrapView(position);获取一个废弃的view，当然这个时候也是没有，所以scrapView为null。执行 mAdapter.getView(position, null, this);这里getView方法的第二个参数传入的是空。通常重写Adapter的getView(int position, View convertView, ViewGroup parent)时候需要先判断convertView是否为空然后进行相应逻辑处理。
2，getView()执行完返回一个子View,然后执行setupChild(View child, int position, int y, boolean flowDown, int childrenLeft,boolean selected, boolean recycled)方法。

etupChild(View child, int position, int y, boolean flowDown, int childrenLeft,boolean selected, boolean recycled)

调用addViewInLayout()方法将obtainView()方法返回的子View添加到了ListView当中。

###最后
这样通过fillDown的while循环，ListView第一页的数据就被加载并且展示出来。由于while循环条件限制，就算有很多数据，最多也只会加载第一页的数据。
